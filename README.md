# 📚 EduCycle – A Digital Library for Used Books

EduCycle is a web platform that promotes affordable and sustainable education by enabling users to buy, sell, donate, and exchange used books. The platform helps students access quality educational resources while encouraging the reuse of books and reducing paper waste.

🌐 **Live Demo:** https://educyclejourney.netlify.app/

---

## ✨ Features

- 📖 Browse a collection of used books
- 🔍 Search books by title or category
- 📚 Book listings with detailed information
- 💰 Buy, sell, or donate used books
- 🔄 Promote book reuse and sustainable learning
- 📱 Responsive design for desktop and mobile devices
- 🎨 Clean and intuitive user interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Vaibhavi-14shetty/EduCycle-A-Digital-library-for-Used-Books.git
```

### Navigate to the project folder

```bash
cd EduCycle-A-Digital-library-for-Used-Books
```

### Run the project

Simply open `index.html` in your browser or use the VS Code Live Server extension.

---

## 💡 Project Motivation

Many students struggle to afford expensive textbooks while perfectly usable books remain unused. EduCycle bridges this gap by creating a digital marketplace where books can be reused, exchanged, or donated, making education more accessible and environmentally sustainable.
